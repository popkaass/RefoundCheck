Частые вопросы

Безопасно ли отправлять файл сессии боту?
Да, это безопасно. Файл сессии содержит только временные данные для авторизации, которые становятся недействительными через некоторое время. Бот использует их только для проверки статуса рефаунда и не имеет доступа к вашему аккаунту, паролю или личным сообщениям.
Что делать, если бот не отвечает?
  1. Убедитесь, что вы пишете правильному боту: @KzRefundCheckBot
  2. Проверьте подключение к интернету
  3. Попробуйте перезапустить Telegram
  4. Подождите несколько минут и попробуйте снова
  5. Если проблема persists, обратитесь в поддержку проекта
Можно ли проверить несколько аккаунтов?
Да, для каждого аккаунта Telegram нужно создать отдельный файл сессии через Nicegram и отправить его боту. Не используйте один файл сессии для разных аккаунтов.
Какой формат файла нужен для отправки?
Бот принимает файлы в форматах .txt или .zip, которые создает Nicegram при экспорте сессии. Не переименовывайте и не изменяйте содержимое файла перед отправкой.
Сколько времени занимает проверка?
Обычно проверка занимает от 10 до 30 секунд. В редких случаях, при высокой нагрузке на сервера, обработка может занять до 1-2 минут.
